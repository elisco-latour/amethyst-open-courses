# question
Which of the following Python built-in types are mutable?

<!-- SEPARATOR -->

# options
- list
- tuple
- dict
- set
- str

<!-- SEPARATOR -->

# explanation
Lists, dicts, and sets can be modified after creation. Tuples and strings are immutable.
