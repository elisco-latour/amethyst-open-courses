# question
Which line correctly defines a class `Dog` that inherits from `Animal`?

<!-- SEPARATOR -->

# options
- class Dog(Animal):
- class Dog inherits Animal:
- class Dog extends Animal:
- def Dog(Animal):

<!-- SEPARATOR -->

# explanation
In Python, parentheses after the class name list the base class(es). `class Dog(Animal):` is the canonical syntax for single inheritance.
