# question
Given `nums = [10, 20, 30, 40]`, what is the value of `nums[-1]`?

<!-- SEPARATOR -->

# options
- 10
- 30
- 40
- IndexError

<!-- SEPARATOR -->

# explanation
Negative indices count from the end. `nums[-1]` is the last element, `40`.
